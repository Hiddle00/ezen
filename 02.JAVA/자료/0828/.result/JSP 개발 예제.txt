필요하면 가져다가 붙여쓰도록





회원관리
index.jsp → 메인 페이지 (로그인 여부에 따라 화면 다르게 보여줌)
join.jsp → 회원가입 입력 폼
joinok.jsp → 회원가입 처리 (DB insert 후 결과 안내)
login.jsp → 로그인 입력 폼
loginok.jsp → 로그인 처리 (세션 저장, 로그인 성공/실패 분기)
logout.jsp → 로그아웃 처리 (세션 삭제 후 메인으로 이동)
modify.jsp → 회원정보 수정 입력 폼
modifyok.jsp → 회원정보 수정 처리 (DB update 후 결과 안내)
delete.jsp → 회원 탈퇴 확인 폼
deleteok.jsp → 회원 탈퇴 처리 (DB delete, 세션 종료)
mypage.jsp → 내 정보 보기 (세션에서 로그인 사용자 정보 가져와 출력)
findid.jsp → 아이디 찾기 (이메일/이름 입력받아 ID 출력)
findpw.jsp → 비밀번호 찾기 (ID/이메일 입력 후 비밀번호 재설정 절차)
findpwok.jsp → 비밀번호 재설정 처리

게시판관리
list.jsp → 게시판 목록 보기 (글 리스트, 페이징)
view.jsp → 게시글 상세 보기
write.jsp → 새 글 작성 폼
writeok.jsp → 글 작성 처리 (DB insert)
edit.jsp → 글 수정 폼
editok.jsp → 글 수정 처리 (DB update)
delete.jsp (게시판 전용) → 글 삭제 확인
deleteok.jsp (게시판 전용) → 글 삭제 처리 (DB delete)
search.jsp → 검색 기능 (키워드로 제목/내용 조회)

기타 보조 JSP
header.jsp / footer.jsp / menu.jsp → include 되는 공통 레이아웃
error.jsp → 예외 상황 발생 시 에러 안내 페이지
sessionCheck.jsp → 세션(로그인 상태) 검증, 로그인 안 하면 로그인 페이지로 redirect
dbconn.jsp → DB 연결 관리용 (JDBC Connection 객체 제공)
