문제 : Person이라는 부모 클래스와 Student라는 자식 클래스를 만드세요. 자식 클래스의 생성자에서 
super()를 사용하여 부모 클래스의 생성자를 명시적으로 호출해야 합니다.

Person 클래스:
name (문자열)과 age (정수) 멤버 변수를 가집니다.
name과 age를 매개변수로 받아 초기화하는 생성자를 작성합니다.
introduce() 메서드를 가지며, 호출 시 "제 이름은 [이름]이고, 나이는 [나이]살입니다."라고 출력합니다.

Student 클래스:
Person 클래스를 상속받습니다.
studentId (문자열) 멤버 변수를 추가로 가집니다.
name, age, studentId를 매개변수로 받는 생성자를 작성합니다.
이 생성자 안에서 **super()**를 사용해 부모 클래스의 생성자를 호출하여 name과 age를 초기화해야 합니다.
study() 메서드를 가지며, 호출 시 "공부를 시작합니다."라고 출력합니다.

main 메서드에서 실행:
Student 클래스의 인스턴스를 이름 "김철수", 나이 20, 학번 "20241234"로 생성합니다.
생성된 인스턴스의 introduce() 메서드와 study() 메서드를 각각 호출하여 결과를 확인합니다.
